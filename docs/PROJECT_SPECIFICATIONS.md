# Spécifications Officielles : Double King Shop (DKS) Hub v3.0

## 1. Vision Stratégique
Le **Double King Shop (DKS)** est le premier Hub Technologique Hybride de la République Démocratique du Congo, situé à Bunia. Il fusionne trois piliers fondamentaux :
- **Commerce de Luxe** : Vente de hardware haute performance (NVIDIA RTX, Starlink, etc.).
- **Éducation d'Élite** : La DKS Academy formant aux métiers de l'IA et de la Blockchain.
- **Infrastructure Business** : Déploiement de réseaux, cybersécurité et maintenance certifiée.

---

## 2. Économie Hybride & Tokenomics
Le Hub repose sur un modèle économique bi-monétaire révolutionnaire :

### A. Consensus Pi Network (GCV)
- **Standard** : Indexation sur le Global Consensus Value (1 π = $314,159).
- **Usage** : Monnaie d'échange prioritaire pour l'acquisition de matériel hardware premium.
- **Interface** : Terminal POS intégré avec génération de QR Code GCV.

### B. Jeton Utilitaire DKST
- **Masse Monétaire** : 50 000 000 DKST (fixée).
- **Distribution** : 65% Pool Communautaire, 20% Fondation/CEO, 15% Équipe.
- **Utilité** : Minage quotidien, Staking (APR 5-12%), paiement des services Academy, et Gouvernance DAO.
- **Halving** : Réduction des récompenses tous les 8M de jetons extraits.

---

## 3. Matrice des Rôles & Permissions (RBAC)
Le système utilise Firebase Auth et Firestore pour gérer quatre niveaux d'accès :

| Rôle | Description | Accès Clés |
| :--- | :--- | :--- |
| **Admin** | Direction Générale | Banque Centrale, Notariat, Gestion Staff, Statistiques Globales. |
| **Vendeur** | Expert Hardware | Gestion du Stock, Assistant IA, SAV Live, Devis Professionnels. |
| **Caissier** | Gestionnaire Flux | Terminal POS, Encaissement Crypto, Impression de Reçus. |
| **Customer** | Membre Élite | Wallet DKST, Academy, Parc Hardware, Testament Numérique. |

---

## 4. Modules Avancés

### A. DKS Academy & Certification
- Inscription dynamique aux cursus IA & Blockchain.
- Calculateur de remises (Groupe & Parrainage).
- Génération automatique de diplômes et certificats de réussite en PDF.

### B. SAV & Laboratoire de Maintenance
- **Support Live** : Chat technique avec partage d'images.
- **Maintenance Consommables** : Gestion des stocks internes (pâte thermique, étain) avec scanner optique.
- **Santé Outillage** : Suivi des cycles d'usure des stations de soudage et calibration.

### C. Sécurité & Héritage (Dead Man's Switch)
- **Signature Élite** : Code PIN à 4 chiffres requis pour tout mouvement de fonds.
- **Digital Heritage** : Transmission automatique des actifs DKST à un héritier désigné après un seuil d'inactivité (30, 90 ou 180 jours).

### D. Gouvernance DAO
- Soumission de propositions stratégiques par l'Admin.
- Vote pondéré par le solde de jetons DKST des membres.

---

## 5. Stack Technique
- **Frontend** : Next.js 14 (App Router), React 18, Tailwind CSS.
- **UI Components** : ShadCN UI, Lucide Icons, Recharts (Analytique).
- **Backend/DB** : Firebase Firestore (Temps réel), Firebase Auth.
- **AI (Genkit)** : Assistant client multilingue, vision par ordinateur, générateur de descriptions marketing.
- **Document Engine** : html2canvas & jsPDF pour la facturation et les certificats.
